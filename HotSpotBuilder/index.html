Quick Tips

• Draw tight boxes, but leave some room for easier dragging.
• In zone mode, use object names exactly as you plan to use them in the scene JSON.
• For between, make sure you also enter valid pairs in the Between Pairs field.
• Foreground masks help you preview whether an item looks natural under tables or behind furniture.
• The preview item is only for visual testing. It is not saved in the scene export.
